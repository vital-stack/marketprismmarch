Verdict banner — redesign preview

Each banner below shows the restructured ticker-page verdict for a different scorecard + primary_label combo. The headline is now the Daily Plays lane + action (from the shared classifier), so a user clicking through from a Daily Plays card sees the same verdict. The structural label (e.g. Fundamental Disconnect) is demoted to a small grey tag. Forensic rebuttal paragraph is unchanged.

What changed vs today

Headline source: was trade_classifications.primary_label. Now classifyLane(scorecard, earnings) — the same function that will be extracted to power Daily Plays lanes, so lane + banner can never drift.

Structural label: demoted from the primary verdict to a small grey tag in the top row. Still visible, still readable, no longer the loudest signal on the page.

Freshness: new "Updated X ago" timestamp from narrative_scorecard.snapshot_date. Small engagement hook for repeat visits.

Forensic rebuttal: unchanged. Still pulls from v_dash_daily_story.forensic_rebuttal and remains the trust/evidence layer.

Fallback: when no lane matches (rare — most names hit at least one), the banner falls back to the old structural-label behaviour so nothing ever renders blank.
